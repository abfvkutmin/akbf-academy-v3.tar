# АКБФ Академия Инвестиций — PWA build v3

## Что нового в третьей версии
- Splash screen при запуске приложения.
- Onboarding-модальное окно с кратким объяснением продукта.
- Более брендированный интерфейс в стиле банковского образовательного сервиса.
- Финальный экран завершения курса.
- Экспорт учебного сертификата в PNG через браузер.
- Сохранены PWA-функции: manifest, service worker, offline shell, install prompt, LocalStorage.

## Структура
- `index.html` — основной интерфейс.
- `styles.css` — стили и брендированный UI.
- `app.js` — логика уроков, виджетов, тестов, onboarding и сертификата.
- `manifest.webmanifest` — манифест PWA.
- `sw.js` — service worker.
- `icons/` — иконки.

## Публикация
1. Загрузить все файлы из папки в репозиторий GitHub.
2. Включить GitHub Pages.
3. Подставить опубликованный URL в iframe-код внутри приложения или в Tilda.

## Замечание
Экспорт сертификата выполняется на клиенте и не требует backend. При желании следующей итерацией можно добавить ввод имени пользователя и персонализацию сертификата.
