# АКБФ Академия Инвестиций — ecosystem-v2

Финальная сборка с многоуровневой структурой курсов. В `app.js` используется массив `COURSES`, внутри которого 6 курсов: старт инвестирования, ИИС-3, облигации, акции, фонды и золото.

Загрузите файлы из архива в корень GitHub Pages репозитория с заменой существующих файлов.
